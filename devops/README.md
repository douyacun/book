# Devops

## 定位
`devops` 用来沉淀 Linux、系统工具、部署发布、网络排障、容器和服务器维护相关内容。

## 收录范围
适合放 Linux 命令、Nginx、Docker、网络诊断、系统性能分析、进程管理、环境配置和故障排查类文章。
不适合放数据库原理、语言语法和纯产品运营内容。

## SEO 方向
主关键词：Linux 教程、Nginx 配置、Docker 使用、服务器排障、系统性能分析。
长尾关键词：负载高排查、epoll 原理、iostat 使用、strace 命令、nginx 反向代理、ssh 配置、libwebp 安装。
常见搜索意图：安装部署、命令速查、异常排查、线上运维、环境搭建。

## 专题建议
可以持续扩展为 Linux 排障手册、Nginx 专题、Docker 部署实践、线上问题处理记录。

## 写作提示
标题优先包含工具名和动作，例如“Nginx 反向代理配置说明”。
正文建议增加命令示例、错误现象、排查路径和最终修复方案，方便被搜索和复用。
