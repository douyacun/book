# Notes

## 定位
`notes` 用来沉淀个人记录、职业思考、资料说明、对外说明文档和不适合归入纯技术分类的内容。

## 收录范围
适合放简历、个人介绍、职业复盘、友情链接说明、声明类文档和杂项记录。
不适合放明确的技术教程或运营方法论，那些应尽量归入更明确的业务分类。

## SEO 方向
主关键词：程序员简历、个人博客说明、职业成长、开发者复盘、技术人经历。
长尾关键词：程序员简历模板、技术博客自我介绍、开发者成长记录、友情链接说明、个人项目介绍。
常见搜索意图：个人资料查看、经验参考、职业路径、对外说明和站点信息了解。

## 专题建议
可以持续整理为职业成长记录、博客建设日志、个人项目档案、公开说明文档几个方向。

## 写作提示
标题可以更自然，但仍建议突出主题，例如“年薪百万的程序员简历”。
正文强调真实经历、结构清晰和可引用性，避免和技术分类内容混杂。
