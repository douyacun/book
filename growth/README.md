# Growth

## 定位
`growth` 用来沉淀内容增长、SEO、平台分发、流量获取和站点运营相关内容。

## 收录范围
适合放 SEO 优化、流量渠道、平台运营、选题策略、内容分发、收录问题和站点增长实验类文章。
不适合放纯技术实现细节，除非文章核心目的是服务增长。

## SEO 方向
主关键词：SEO 优化、内容增长、流量获取、站点运营、平台分发。
长尾关键词：Google 收录、sitemap 优化、站内 SEO、公众号引流、小红书引流、技术博客增长。
常见搜索意图：如何获取流量、如何提升收录、平台运营经验、内容结构优化。

## 专题建议
可以围绕技术博客 SEO、内容分发矩阵、平台引流策略、搜索收录优化持续沉淀专题。

## 写作提示
标题建议带结果感和平台名，例如“技术博客如何做 Google SEO 收录优化”。
正文要尽量量化指标、拆分动作和复盘结果，方便后续在社媒平台二次分发。
