# Frontend

## 定位
`frontend` 用来沉淀前端开发、React、SSR、跨端集成、桌面 UI 和用户侧实现相关内容。

## 收录范围
适合放 React、前端工程化、渲染机制、组件实践、SSR、Markdown 渲染、界面组件和客户端交互类文章。
不适合放服务器部署、数据库原理和纯后端性能分析内容。

## SEO 方向
主关键词：React 教程、前端工程化、SSR、Markdown 渲染、前端性能优化。
长尾关键词：React SSR、react-markdown、cookie 跨域、Twitter 分享、QTableWidget、Adsense 集成。
常见搜索意图：组件实现、渲染方案、问题排查、第三方集成、功能教程。

## 专题建议
可以继续沉淀 React 工程实践、SSR 内容站、第三方分享集成、桌面端界面开发等系列。

## 写作提示
标题要尽量点出框架名和问题点，例如“React SSR 中如何接入 Adsense”。
正文建议用“场景、实现、坑点、兼容性、总结”的结构，方便平台分发时裁剪。
