# PDF

## 定位
`pdf` 用来沉淀 PDF 工具、OCR、文档处理、去水印、格式转换和相关产品运营内容，是当前最贴近业务变现和搜索流量的分类。

## 收录范围
适合放 PDF 去水印、批量处理、OCR 识别、图片转文字、格式转换、产品教程、下载说明和常见问题解决类文章。
不适合放底层数据库原理和系统架构思考，这些应该分别进入 `database` 或 `architecture`。

## SEO 方向
主关键词：PDF 去水印、PDF 编辑、OCR 识别、PDF 转换、PDF 工具、图片识别文字。
长尾关键词：WPS PDF 去水印、批量替换 PDF 文字、扫描全能王去水印、亿图图示去水印、PDF 加白边、图片文字识别为 Word。
常见搜索意图：具体操作教程、工具下载、问题解决、功能对比、批量处理方案。

## 专题建议
适合持续做 PDF 去水印专题、OCR 与文档识别、PDF 工具箱教程、文档处理产品运营几个方向。

## 写作提示
标题尽量直接命中搜索行为词，例如“如何去除 PDF 水印”。
正文建议结论前置、步骤明确、截图充分，并加入相关文章和工具对比链接。
