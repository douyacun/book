centos7安装php7.4

- 安装及安装扩展

```
yum install libxml2-devel sqlite-devel libcurl-devel oniguruma-devel libpng-devel libjpeg-devel freetype-devel libzip-devel openssl-devel -y
```

```
./configure --prefix=/usr/local/php82 --with-config-file-path=/etc --with-mhash --with-openssl --with-mysqli=mysqlnd --with-pdo-mysql=mysqlnd --with-iconv --with-zlib --disable-debug --disable-rpath --enable-shared --enable-bcmath --enable-shmop --enable-sysvsem --enable-gd --with-jpeg --with-freetype --enable-mbregex --enable-mbstring --enable-ftp --enable-pcntl --enable-sockets --enable-soap --without-pear --with-gettext --enable-session --with-curl  --enable-opcache --enable-fpm --with-fpm-user=php --with-fpm-group=php --without-gdbm --enable-fast-install 
```

- 查看php版本
    php74 -v

- 配置文件 php.ini 和 php-fpm.conf

php.ini-production

```
cp php.ini-production /etc/php.ini
```

- 启动和添加开机启动

```
cd php-8.2.7/sapi/fpm

cp php-fpm.service /etc/systemd/system/

# 启动php-fpm
systemctl start php-fpm

# 查看是否启动
systemctl status php-fpm
```

- 修改用户

```
vi /etc/opt/remi/php74/php-fpm.d/www.conf
user = apache
group = apache
修改为
user = nginx
group = nginx
```
